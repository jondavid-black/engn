# Digital Engine (digengn)

Digital Engine is a comprehensive suite of intelligent tools designed for modern engineering and program management. It provides a robust foundation for building, architecting, and managing complex systems through a collection of specialized engines.

## Core Components

- **Engn**: Core CLI tool and shared libraries providing foundational utilities.
- **SysEngn**: Model-Based System Engineering (MBSE) and User Experience (UX) design tool.
- **ProjEngn**: Program Management tool for managing schedules and deliverables.

## Getting Started

Refer to the [documentation](docs/index.md) for detailed information on how to use the Digital Engine suite.
